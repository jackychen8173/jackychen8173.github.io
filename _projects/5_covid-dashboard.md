A two-page Power BI dashboard visualizing COVID-19 trends across British Columbia using BCCDC open data, designed as a portfolio project to demonstrate data analytics and visualization skills.

## Overview

This dashboard provides an interactive view of COVID-19 metrics in British Columbia, including case trends, regional breakdowns, and temporal analysis. The project showcases ability to work with public health data and create actionable visualizations.

## Tech Stack

- **Power BI Desktop** - Dashboard development
- **BCCDC Open Data** - Public health data source
- **DAX** - Custom calculations and measures

## Dashboard Features

**Page 1: Overview**
- Time-series trends of cases, hospitalizations, and deaths
- Key performance indicators (KPIs)
- Date range filtering

**Page 2: Regional Analysis**
- Geographic breakdown by health authority
- Comparative metrics across regions
- Interactive drill-down capabilities

## Data Processing

- Cleaned and transformed raw BCCDC data
- Created calculated measures for rates and trends
- Designed data model for efficient querying

## Visualizations

- Line charts for temporal trends
- Bar charts for regional comparisons
- Cards for KPI metrics
- Slicers for interactive filtering

## Deliverables

- Published dashboard with screenshots
- `.pbix` file for reproducibility
- Documentation of data sources and methodology

## Purpose

This project demonstrates:
- Data analysis and visualization skills
- Power BI proficiency
- Understanding of public health metrics
- Ability to communicate insights through visual design

## What I Learned

- Power BI dashboard design and development
- Working with time-series public health data
- DAX formula creation
- Visual design principles for data communication
- Publishing and sharing BI reports
